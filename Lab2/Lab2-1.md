## Computer Programming for Computer Engineer 

# CODE
```c++
#include <stdio.h>

int main() {
    int N = 0 ;

    if(scanf("%d" , &N) != 1 ) {
        return 1 ;
    }

    for (int i = 0; i < N; i++) {
        printf("Hello Loop!\n");
    }
    
    return 0 ;
}
```
